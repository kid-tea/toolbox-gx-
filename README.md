# 工具箱

WPF .NET 8.0 多功能系统工具箱。

## 运行要求

- Windows 10/11 x64
- [.NET 8 桌面运行时](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)

## 更新日志

### v1.0.3
- 磁盘分析新增 WizTree 风格矩形树图（Squarified Treemap），替换饼图
- 文件按扩展类型着色，点击联动树节点展开定位
- C盘清理新增年龄过滤、界面布局优化
- 批量重命名新增整体重命名和格式/扩展名规则
- 注册表清理增强备份校验、减少误判、安全默认选项
- 内存释放修复权限和命令错误
- UI 像素对齐优化

### v1.0.2
- 磁盘空间分析：文件夹树与空间分布区域可拖拽调整大小
- 文件节点悬停显示完整路径
- 修复独立弹窗 StaticResource 资源查找失败崩溃（L2/L3确认框、悬浮任务面板）
