# Toolbox 工具箱

基于 WPF + .NET 8 构建的 Windows 本地多功能系统工具箱。

## 运行要求

- Windows 10/11 x64
- [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)

## 下载

正式发布文件请在 GitHub Releases 中下载：

- [Toolbox.exe](https://github.com/kid-tea/toolbox-gx-/releases/download/v1.0.5/Toolbox.exe)

## 更新历史

### v1.0.5

发布日期：2026-07-05

#### 使用体验修复

- 修复设置页主题保存后重启丢失的问题。
- 修复进入设置页时自动触发保存的问题。
- 优化重复文件查找的默认扫描范围：默认只选择本地固定磁盘，避免用户目录和桌面与 C 盘重复扫描。
- 将重复文件查找的“选择路径”改为“智能选择”，点击后不再弹窗，只恢复推荐磁盘选择。
- 修复重复文件查找页“智能选择”按钮文字被裁切的问题。
- 将所有待优化功能集中到“调试功能”专区，正式功能导航更清晰。

#### 构建与发布

- 版本号更新为 `1.0.5`。
- 发布文件更新为 `publish/Toolbox.exe`。
- `obj` 中间构建目录不再作为发布内容提交。
- 保持 `bin`、`publish` 本地构建输出可用，Release 附件使用 `publish/Toolbox.exe`。

### v1.0.4

发布日期：2026-07-05

#### 主要更新

- 新增多套主题风格，并可在设置页切换。
- 优化左侧工具栏交互，支持手动收起和展开。
- C 盘清理新增更保守的年龄过滤逻辑，默认 24 小时且默认不勾选。
- 批量重命名新增整体重命名和格式化重命名能力。
- 优化磁盘分析 Treemap，小文件以块状聚合展示。
- 改进注册表清理备份校验和空注册表键一键全选。
- 修复强制删除、内存释放、文件粉碎等功能中的可用性问题。

AI Agent 检验仍属于 `2.0.0` 预览/调试功能，不作为 `1.0.5` 正式功能发布。
