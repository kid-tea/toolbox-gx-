# 工具箱

WPF .NET 8.0 多功能系统工具箱，16 个实用功能模块。

## 功能模块

| 分类 | 功能 | 说明 |
|------|------|------|
| 💾 存储清理 | 磁盘空间分析 | 扫描磁盘目录树，大文件展示，右键打开/删除 |
| 💾 存储清理 | C盘清理 | 安全/推荐/高级三级清理，L3 确认保护 |
| 💾 存储清理 | 大文件扫描 | 全盘扫描超大文件 |
| 💾 存储清理 | 重复文件查找 | 按内容查找重复文件 |
| 📁 文件操作 | 强制删除 | 解除占用后强制删除文件 |
| 📁 文件操作 | 文件粉碎 | 多次覆写防止恢复 |
| 📁 文件操作 | 批量重命名 | 规则批量重命名文件 |
| ⚙️ 系统管理 | 开机启动项管理 | 管理 Windows 启动项 |
| ⚙️ 系统管理 | 右键菜单管理 | 清理/管理右键菜单 |
| ⚙️ 系统管理 | 注册表清理 | 扫描并修复注册表问题 |
| ⚙️ 系统管理 | 内存释放 | 释放系统内存 |
| ⚙️ 系统管理 | 定时关机 | 定时/空闲自动关机 |
| 🎯 日常效率 | 截屏 | 全屏/区域/窗口/延迟截图，支持标注 |
| 🎯 日常效率 | 取色器 | 屏幕取色工具 |
| 🎯 日常效率 | 窗口置顶 | 任意窗口置顶 |

## 运行要求

需要安装 [.NET 8 桌面运行时](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)

## 版本历史

### v1.0.2
- 添加应用自定义图标
- 磁盘空间分析：文件夹树与空间分布区域可拖拽调整大小
- 文件节点悬停显示完整路径
- 更新 publish 分发文件

### v1.0.1
- 磁盘空间分析功能正式上线（移除调试限制）
- 修复 ProgressBar 只读属性 TwoWay 绑定崩溃
- 磁盘扫描改为栈式迭代，避免深层目录 StackOverflow
- 目录下展示最大的 20 个文件，支持右键操作
- 右键菜单：打开文件所在位置 / 删除文件（系统保护路径禁止删除）
- 添加全局 UI 未处理异常捕获，防止闪退
- 修复 C盘清理 CheckBox 选中状态递归事件循环
- 修复全屏截图快捷键窗口最小化时启动问题

### v1.0.0
- 首次发布，16 个功能模块
