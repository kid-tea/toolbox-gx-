# Toolbox 工具箱

WPF + .NET 8 构建的 Windows 多功能系统工具箱。

## 运行要求

- Windows 10/11 x64
- [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)

## 下载

正式发布文件请在 GitHub Releases 中下载：

- [Toolbox.exe](https://github.com/kid-tea/toolbox-gx-/releases/download/v1.0.5/Toolbox.exe)

## 功能模块

| 分类 | 功能 | 说明 |
|------|------|------|
| 存储清理 | C 盘清理 | 安全扫描并清理常见缓存、临时文件和可选风险项 |
| 存储清理 | 磁盘空间分析 | 扫描目录占用，使用矩形树图展示空间分布 |
| 存储清理 | 大文件扫描 | 查找超大文件 |
| 存储清理 | 重复文件查找 | 按文件内容查找重复文件 |
| 文件操作 | 文件强制删除 | 解除占用后删除难删文件 |
| 文件操作 | 文件粉碎 | 覆写删除，降低恢复可能 |
| 文件操作 | 批量重命名 | 支持前后缀、整体重命名和格式化重命名 |
| 系统管理 | 开机启动项管理 | 管理 Windows 启动项 |
| 系统管理 | 右键菜单管理 | 管理右键菜单项目 |
| 系统管理 | 注册表清理 | 扫描并清理低风险注册表问题，支持备份校验 |
| 系统管理 | 内存释放 | 清理待机列表、系统缓存等内存项 |
| 系统管理 | 定时关机 | 定时或空闲自动关机 |
| 日常效率 | 取色器 | 屏幕取色工具 |
| 日常效率 | 窗口置顶 | 任意窗口置顶 |
| 调试功能 | AI Agent 检验 | 2.0.0 预览功能，默认放入调试专区 |
| 调试功能 | 截屏、任务 | 待优化功能，默认放入调试专区 |

## 更新历史

### v1.0.5

发布日期：2026-07-05

#### 使用体验修复

- 修复设置页主题保存后重启丢失的问题。
- 修复进入设置页时自动触发保存的问题。
- 优化重复文件查找的默认扫描范围：默认只选择本地固定磁盘，避免用户目录和桌面与 C 盘重复扫描。
- 将重复文件查找的“选择路径”改为“智能选择”，点击后不再弹窗，只恢复推荐磁盘选择。
- 修复重复文件查找页“智能选择”按钮文字被裁切的问题。
- 将所有待优化功能集中到“调试功能”专区，正式功能导航更清晰。

#### 构建与发布

- 版本号更新为 `1.0.5`。
- 发布文件更新为 `publish/Toolbox.exe`。
- `obj` 中间构建目录不再作为发布内容提交。
- Release 附件使用 `publish/Toolbox.exe`。

### v1.0.4

发布日期：2026-07-05

#### 主要更新

- 新增第二套深色仪表盘主题，可在设置页与经典风格之间切换。
- 优化左侧工具栏交互，支持手动收起和展开，不再因鼠标划过自动弹出。
- 左侧导航默认只显示一级分类，默认展开“存储清理”分类。
- 全局滚动条改为细条样式，鼠标悬停时再变宽。
- 修复部分主题下控件底色、边框和文字颜色冲突的问题。

#### C 盘清理

- 删除前年龄过滤改为用户主动勾选后才生效。
- 年龄过滤默认取消勾选，默认阈值从 48 小时调整为 24 小时。
- 扩展安全扫描项，提升可发现的缓存和临时文件范围。
- 优化清理项展示，减少重复项和同步项造成的误导。
- 补充风险项副作用和建议使用频率说明。

#### 文件与磁盘工具

- 批量重命名新增整体重命名和格式化重命名能力。
- 优化磁盘分析 Treemap，小文件以块状聚合展示，减少文字挤压。
- 改进 Treemap 与文件夹树联动，点击图形时可定位对应文件或文件夹。
- 修复强制删除按钮无法点击的问题。
- 检查并保持文件粉碎相关交互可用。

#### 注册表清理

- 增强注册表备份可靠性。
- 新增备份校验，恢复前可发现备份是否被修改或损坏。
- 空注册表键新增一键全选能力。
- 默认策略更保守，降低误删高风险注册表项的概率。

#### 内存释放

- 修复“待机列表”和“系统缓存”操作显示错误的问题。
- 补充相关操作的副作用说明和建议使用频率。

#### 说明

- 推荐发布文件：`publish/Toolbox.exe`
- 运行环境：Windows + .NET 8 Desktop Runtime
- AI Agent 检验功能仍处于调试/待优化阶段，不作为 `1.0.4` 正式功能发布。
