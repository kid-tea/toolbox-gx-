# Toolbox 工具箱

WPF + .NET 8 构建的 Windows 多功能系统工具箱。

## 运行要求

- Windows 10/11 x64
- [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)

## 下载

正式发布文件请在 GitHub Releases 中下载：

- `Toolbox.exe`

## v1.0.4 更新内容

### 界面与交互

- 新增第二套深色仪表盘主题，可在设置页切换。
- 优化左侧工具栏交互，支持手动收起和展开，避免鼠标划过时误弹出。
- 左侧导航默认只显示一级分类，默认展开“存储清理”分类。
- 全局滚动条改为细条样式，鼠标悬停时再变宽，减少界面占用。
- 修复部分主题下控件底色和文字颜色接近、难以阅读的问题。

### C 盘清理

- 删除前年龄过滤改为用户主动勾选后才生效。
- 年龄过滤默认取消勾选，默认阈值从 48 小时调整为 24 小时。
- 优化清理项说明，补充风险、副作用和建议使用频率。
- 减少重复扫描项和同步显示造成的误导。

### 文件与磁盘工具

- 批量重命名新增整体重命名和格式化重命名能力。
- 优化磁盘分析 Treemap，小文件以块状聚合展示，减少文字挤压。
- 强化 Treemap 与文件夹树联动，点击图形时可定位对应文件或文件夹。
- 修复强制删除按钮无法点击的问题，并检查文件粉碎相关交互。

### 注册表清理

- 增强注册表备份与恢复前校验。
- 空注册表键新增一键全选能力。
- 默认策略更保守，降低误删高风险注册表项的概率。

### 内存释放

- 修复“待机列表”和“系统缓存”操作显示错误的问题。
- 补充相关操作的副作用说明和建议使用频率。

## 版本说明

- `v1.0.4` 只包含正式工具箱功能增强与修复。
- AI Agent 检验功能仍处于调试/待优化阶段，不作为 `v1.0.4` 正式功能发布。
