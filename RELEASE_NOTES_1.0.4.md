# Toolbox 1.0.4

发布日期：2026-07-04

## 主要更新

- 优化整体 UI 体验，新增更柔和的深色仪表盘主题，可在设置页与经典风格之间切换。
- 改进左侧工具栏交互：工具栏支持手动收起/展开，不再因鼠标划过自动弹出；默认只展开存储清理分类。
- 全局滚动条改为细条样式，鼠标悬停时滑块变宽，减少界面占用但保留易用性。
- 修复仪表盘主题下部分原生控件底色、边框和文字颜色冲突问题，包括 Tab、GroupBox、ComboBox、TextBox、Button、CheckBox、Slider。

## 存储清理

- C 盘清理加入更明确的删除年龄过滤逻辑：用户勾选后才按年龄条件清理，未勾选时不启用该过滤。
- 扩展安全扫描项目，提升可发现的缓存和临时文件范围。
- 优化清理项展示，减少重复项目和误导性同步项。
- 强化危险项说明，补充副作用和建议使用频率，降低误清理风险。

## 磁盘分析

- 优化磁盘占用可视化，将小文件更适合地聚合为块状布局，减少长条挤压文字的问题。
- 改进块状图与文件夹树的联动，点击图形时能更准确定位对应文件或文件夹。
- 强化文字可读性和图形层级，让大文件、文件夹和小文件聚合更容易区分。

## 批量重命名

- 在原有前后缀能力基础上，补充整体重命名和格式化重命名能力。
- 改进重命名规则的实用性，适合批量整理文件名。

## 内存释放

- 修复待机列表和系统缓存按钮调用逻辑混用的问题。
- 修复清空待机列表可能出现权限错误的问题，调用前会尝试启用必要权限。
- 补充待机列表、系统缓存等操作的副作用说明和建议使用频率。

## 注册表清理

- 增强注册表备份可靠性，删除前备份信息更完整。
- 新增备份校验，恢复前可发现备份是否被修改或损坏。
- 修复部分启动项残留无法正确备份/恢复的问题。
- 修复前会再次验证问题是否仍然存在，降低扫描后状态变化导致误修复的风险。
- 空注册表键和高风险 COM/OLE 项默认更保守，并提供一键全选分类能力。
- 空键扫描跳过 Microsoft、Classes、Windows 等高风险区域，降低误删风险。

## 说明

- 推荐发布文件：`publish/Toolbox.exe`
- 运行环境：Windows + .NET 8 Desktop Runtime
